phase: KYPHOTIC - Echelon

Tags: Blueprints UE4 Multiplayer Game Windows

Links: Download (.zip) (Windows)

project: KYPHOTIC - Echelon, or p:KE, is a a cyberpunk-themed twin-stick shooter created in Unreal Engine 4. The development team for p:KE consisted of four students, myself included, who sought out to create a fully functional game from scratch over the course of the sixteen-week semester. The team heavily utilized UE4's blueprint system in order to speed up development and to learn more about the engine's capabilities. The goal of the project was to gain experience and familiarity with the tools offered in Unreal Engine 4.

The game features up-to-four-player multiplayer splitscreen, multiple enemy types each with varying attacks, and two levels to play through: the first being more traditional, and the second being a boss battle. p:KE supports both mouse and keyboard input, as well as controllers.

Duane's Great Adventure (Pong)

Tags: C++ Proprietary Engine Singleplayer Game Windows

Links: Download (.zip) (Windows)

Created for my Video Game Programming course using Webfoot Technologies' Frog Libraries, "Duane's Great Adventure" is a Pong clone that is aesthetically inspired by 80s vaporwave and a short-lived internet fad, Duane. The goal of the project was to simply recreate Pong, with extra credit being awarded to students who went above and beyond to add extra assets to their project.

The game is entirely coded in C++ using the aforementioned Frog Libraries from Webfoot Technologies. A majority of the art assets used were created from scratch.

SEGA Dreamcast OpenGL Demo

Tags: C++ OpenGL 3D Dreamcast Demo

Links: Source (GitHub) CDI (Dreamcast)

A small demostration that renders and animates 3D objects on the SEGA Dreamcast hardware using OpenGL and C++. This demo is written entirely in C++ and hooks into the KallistiOS Dreamcast framework. This project also makes use of kazade's "GLdc" OpenGL Dreancast libraries.

When I had initially discovered the SEGA Dreamcast homebrew scene, I was enchanted by the idea of developing something that could run on real console hardware. After taking some time to learn more about programming, I decided to take up the challenge of learning not only the KOS framework, but also OpenGL.

BunnBot

Tags: Python Picarto Plugins Chat Bot

Links: Source (GitHub)

BunnBot is an automated chat bot designed to assist in administrating and entertaining chat rooms for Picarto.TV's livestreaming service. BunnBot is written in Python, and features a plugin system that allows third-party developers to create their own custom functionality.

The bot uses Picarto.TV's webhooks to connect to a websocket and listens for any incoming messages from the server. Developers can choose to have their plugins act on these messages, such as running any new chat messages through a swear filter, checking to see if a user has input a special command, or by greeting new users as they enter the chat room.
